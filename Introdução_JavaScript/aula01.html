Introdução ao JavaScript

Variáveis, constantes, prompt(), alert() e console.log()

código fonte
aula01.js 
// CONSTANTES — valor fixo, não pode ser alterado
const ANO_ATUAL      = 2026;           // número
const PI             = 3.14159;        // decimal
const ATIVO          = true;           // booleano

// VARIÁVEIS — valor pode mudar
let nome;
let idade;
let cidade;

// ENTRADA — prompt() retorna uma string
nome   = prompt("Digite seu nome:");
idade  = prompt("Digite sua idade:");
cidade = prompt("De qual cidade você é?");

// PROCESSAMENTO — converte string → número
let idadeNumero  = Number(idade);
let anoNascimento = ANO_ATUAL - idadeNumero;

// SAÍDA — alert() exibe caixa de diálogo
alert("Olá, " + nome + "! Nasceu em " + anoNascimento);

// SAÍDA — console.log() imprime no console (F12)
console.log("Nome: "   + nome);
console.log("Cidade: " + cidade);
console.log("typeof nome:", typeof nome);      // "string"
console.log("typeof idadeNumero:", typeof idadeNumero); // "number"
conceitos

const — constante

const PI = 3.14;
const ATIVO = true;

Valor não pode ser reatribuído. Use para dados fixos como versões, configurações e valores matemáticos.

let — variável

let nome; // undefined
nome = "João"; // atribuição

Valor pode ser alterado. Use para dados que mudam: entrada do usuário, resultados de cálculos.

prompt()

let n = prompt("Nome:");
// retorna: string ou null

Exibe uma caixa de entrada de texto. Sempre retorna string — use Number() para converter.

alert() e console.log()

alert("Mensagem");
console.log(variavel);

alert mostra caixa de diálogo na tela. console.log imprime no console do navegador (F12).
