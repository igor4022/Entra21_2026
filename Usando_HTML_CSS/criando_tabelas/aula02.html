4. Métodos GET e POST

Método GET

O método GET envia os dados pela URL.

Exemplo:

site.com?nome=joao
  • Mais usado em pesquisas
  • Dados visíveis na URL
  • Menor segurança

Método POST

O método POST envia os dados ocultos no corpo da requisição.

  • Mais seguro
  • Usado para cadastros e login
  • Dados não aparecem na URL
