JavaScript — Aula 04

Arrays & Objetos

Estruturas de dados, métodos e iteração

Array

Frutas

Declaração e acesso por índice

let frutas = ["Maçã", "Banana", "Laranja"];
frutas[0] // → Maçã
frutas[1] // → Banana
frutas[2] // → Laranja
Métodos

push · pop · length

Manipulando arrays dinamicamente

alunos.push("Marcos") // adiciona
alunos.pop()          // remove último
alunos.length         // → 2
Objeto

Propriedades & Mutação

Acesso, alteração e adição de propriedades

let aluno = { nome:"João", idade:18, curso:"JavaScript" };
aluno.idade = 19;          // altera
aluno.cidade = "Blumenau"; // adiciona
Método

Função no Objeto

Objetos podem conter funções

let aluno_teste = {
  nome: "Maria",
  apresentar: function() {
    console.log("Olá, meu nome é " + this.nome);
  }
};
aluno_teste.apresentar();
Array de Objetos

Turma de Alunos

Cada posição do array é um objeto completo

alunos_obj[0].nome  // → Ana
alunos_obj[1].idade // → 18
Iteração

for · forEach

Duas formas de percorrer arrays

for clássico

for(let i = 0; i < alunos.length; i++) {
  console.log(alunos[i]);
}

forEach moderno

alunos.forEach(function(aluno) {
  console.log(aluno);
});
